NAME

mime_gif - encode a single gif by reading data different ways

SYNOPSIS

Usage: 

    mime_gif /path/to/some.gif

DESCRIPTION

Encode a multipart message where each part contains the same GIF file, but where the GIF file has been read-in in different ways. The subject line of each part will tell you how the GIF was read.
