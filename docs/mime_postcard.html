NAME

mime_postcard - output a multipart/alternative message

SYNOPSIS

Usage: 

    mime_postcard /path/to/some/graphic.jpg to@addr.com

You can point it at a .gif file as well. The special address "-" just causes the message to go to STDOUT.

DESCRIPTION

This send a mesasge both as HTML and plain text. I use "Data"; you would probably use "Path".
